태그는 열림태그와 닫힘태그가 존재한다. 이것은 /로 구분하고 태그를 작성할땐 <>를 이용한다
글앞에 필요한 태그를 작성하고 해당 태그가 끝나는 지점에 태그앞에 /를 표시해준다

1. 진하게 (strong)
2. 밑줄 (u>
3. 제목글 (h1~h6)
4. 줄바꿈 (br) -> 닫는태그 없음
5. 단락(p)
6.이미지불러오기(img)
7.목차(li)
8.링크(a)

속성



img의 속성 두가지
1. src 사진의 링크를 넣는 곳 src="사진링크(주소)" 로 사용하면된다
2. width 사진의 크기 조절. width=1~100%로 조절할 수 있다
source의 줄임말인 src를 붙혀서 그림주소를 넣어준다
그림 주소뒤에 width=%를 넣어주면 그림크기를 조절할 수 있다.

  • a

ol은 ordered list의 약자 ->숫자 목록
ul은 unordered list의 약자 -> 점 목록
목차를 지정할때는 li를 사용하면 되고
이것의 부모태그로 ol이나 ul를 사용하면 구분이 된다.

문서의 기본

  1. title : 웹페이지의 제목
    영어가 아닌 언어일 경우 형식이 그럼
    그럴때는 웹페이지가 편집기의 형식과 같이 읽을 수 있는 태그를 사용하면됨
  2. body : 본문 태그
    본문이 시작할때 해당 태그를 열고 끝나면 /를 이요하여 닫는다
  3. head : 본문을 설명하는 태그
    title 이나 meta charset 등 태그적으로 본문을 설명할때 사용된다.
  4. html : body,head를 감싸는 태그
  5. !doctype html : html 문서 제일 처음에 넣는 태그
=> 웹페이지 구조는 이런식이다
!doctype html 제일위
그아래 html 태그
그아래 head, body태그
